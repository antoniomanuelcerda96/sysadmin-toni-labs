# OverTheWire Bandit Write-ups

This repository contains my personal write-ups while completing the **OverTheWire Bandit** wargame.

The objective of this project is not only to solve each challenge, but also to understand the Linux concepts behind every level and document them as study notes for future reference.

Each write-up includes:

- Challenge objective
- Problem analysis
- Solution
- Command explanation
- Real-world system administration usage
- Skills developed


## Technologies & Commands

Throughout this project I practice and document Linux concepts such as:

- SSH
- File system navigation
- Hidden files
- File permissions
- File inspection
- Text processing
- Binary analysis
- Unix pipelines

Main commands learned:

- `ssh`
- `ls`
- `cd`
- `cat`
- `file`
- `find`
- `grep`
- `sort`
- `uniq`
- `strings`
